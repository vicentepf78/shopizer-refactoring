# ADR-004: Clients Strangler com RestTemplate

## Status

Aceito

## Data

2026-07-18

## Contexto

A Onda 1 precisa de clients HTTP para (1) adapters Strangler do monolito → serviços reference/tax e (2) tax-service → reference-service para resolução de country/zone. O codebase não tem precedente de Feign/WebClient; os testes já usam `TestRestTemplate`. STATE AD-005 escolheu RestTemplate.

## Decisão

Usar Spring `RestTemplate` configurado como `@Bean` para todos os clients HTTP da Onda 1. Configurar base URLs via properties (`wave1.reference-service.base-url`, `wave1.tax-service.base-url`). Timeout padrão do client 5s (`wave1.http.client.timeout-ms`). Sem Spring Cloud / OpenFeign / Eureka na Onda 1 (OQ-06).

## Alternativas consideradas

### Alternativa 1: OpenFeign / Spring Cloud

- **Descrição**: Clients declarativos + discovery opcional.
- **Prós**: Bindings de interface mais limpos; ecossistema para discovery.
- **Contras**: Nova dependência de stack; sem precedente; mais pesado para um piloto.
- **Por que rejeitada**: AD-005 / YAGNI; discovery adiado para Onda 2+.

### Alternativa 2: WebClient (reativo)

- **Descrição**: Client HTTP non-blocking.
- **Prós**: Stack moderna; melhor concorrência sob carga.
- **Contras**: Puxa padrões reativos para um monolito servlet; curva mais íngreme para os adapters.
- **Por que rejeitada**: Inconsistente com os padrões imperativos de teste/client existentes.

## Consequências

### Positivas

- Um único estilo de client entre adapters e tax→reference.
- Fácil de mockar com `MockRestServiceServer` / WireMock.

### Negativas

- Mais boilerplate que Feign.
- Propagação manual de headers (JWT, correlation id).

### Riscos

- Timeout mal configurado causando falhas em cascata — default 5s + mapeamento estruturado para 503; ajustar por ambiente.

## Notas de implementação

- Propagar `X-Correlation-Id` (gerar UUID se ausente).
- O adapter de tax encaminha `Authorization` e query params `store`/`lang`.
- Connection refused / timeout → HTTP 503 no BFF; sem fallback silencioso in-process quando o Strangler estiver habilitado.
